# API wrapper calls

This folder provides thin wrapper functions for every API resource and method.
They are not required at runtime, but act as a stable call surface for future
features.

Resources currently exposed here:

- `auth`
- `settings`
- `user`
- `room`
- `roomBot`
- `chat`
- `playlist`
- `source`
- `friend`
- `shop`
- `billing`
- `system`
